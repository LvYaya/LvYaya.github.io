数组中一些常用的方法

发表于 2017-09-18 | 分类于 js

splice()

2个参数splice(开始位置,要删除的项数)
3+个参数splice(开始位置,0,要插入的项) 0：就是不删除一项，直接插入
3+个参数splice(开始位置,1,要插入的项) 1:删除一项，再插入，也就是替换

slice()

slice(n)一个参数：只有一个参数，返回的是从指定项到到当前数组末尾的所有项
slice(n,m)有两个参数，（开始，结束）；返回的是从开始到结束的项，不包括最后一项

indexOf()和lastIndexOf()

indexOf(n)&&lastIndexOf(n):一个参数返回查找项所在的位置,没有返回-1”
indexOf(n,m)两个参数(要查找的项，开始查找的位置)
lastIndexOf(n,m)两个参数(要查找的项，开始查找的位置(从后向前开始数),所以如果查找的项有多项，返回的是后面一个项的位置)

迭代方法every(),some(),filter(),map(),forEach()

every(),some(),filter(),map(),forEach()
对数组中的每一项运行给定函数；

every()：每一项都满足返回true
some():有一项满足都返回true；
filter():可以在函数中筛选，返回满足条件的项，返回的是数组
map():对每一项进行操作，返回的是数组
forEach():遍历每一项，可以执行某些操作

归并方法总结：

push(),pop(),push(),shift()

栈：push(),pop()
队列：push(),shift()
反向队列：unshift()推入到第一项,pop()弹出最后一项;

sort()排序,reverse()翻转

注意sort()排序，并不会正常排序

concat(),join()

其他

toSting()
